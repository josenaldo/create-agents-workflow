# Write ADR

1. Check `docs/specs/adr/` for the next ADR number.
2. Copy `docs/specs/adr/_TEMPLATE.md` → `docs/specs/adr/ADR-NNN-{slug}.md`.
3. Fill in all sections: Context, Decision, Consequences (positive, negative, risks).
4. Set status to `proposed` (or `accepted` if already approved).
5. Reference related issues/PRs if any.
